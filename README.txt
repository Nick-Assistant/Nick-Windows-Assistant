1. скачай ollama.
2. скачай подходящую модель для своей видеокарты (по весу не более чем обьем видеопамяти минус 3-4 Гб)
3. параллельно с моделью .gguf создай файл без расширения под названием Modelfile. Напиши в нем одну строку и сохрани: "FROM ./{точное имя модели}.gguf"
4. в этой же папке нажми на путь расположения и напиши там cmd для открытия терминала.
5. напиши "ollama create {придумай название модели (лучше на английском и без пробела)} -f Modelfile"
6. enter
7. внутри Nick\Nick-Assistant-main\modules\open_interpreter.py замени "my-assistant" на только что придуманное название.
8. установи Python 3.10-3.12 c пометкой "Add to PATH"
9. в модельке "install_and_run.bat" для загрузок библиотек.



10. все готово, дальнейшие запуски через "NICK Assistant.bat"
